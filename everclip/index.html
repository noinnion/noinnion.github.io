Clipboard Monitor

Text copied to clipboard will be automatically add to EverClip. It also recognizes urls and downloads the corresponding data.

[image src="/wp-content/uploads/clipboard_monitor.png" alt="Clipboard Monitor"]
[/column]

[column type="one-third"]

Text Expander

Do you have phrases which you write very often. With Text Expander you can create a snippet with an abbreviation. Typing this abbr. in any text box will replace it with the saved phrase.

[image src="/wp-content/uploads/text_expander.png" alt="Text Expander"]
[/column]

[column type="one-third" last="true"]

Web Clipper

You can clip web information from webpages and save the content to Evernote. It never was easier to save website content.

[image src="/wp-content/uploads/web_clipper.png" alt="Web Clipper"]
[/column]
