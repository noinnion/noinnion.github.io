Run gReader on ChromeOS or Chrome in Windows, MacOSX and Linux

20 Sep 2014

Google has officially made it possible to run Android apps on Chrome OS devices. Developer Vladikoff made ArChon, a customized version of the Android Runtime for Chrome, which loads up as a standard manual Chrome extension.

Github: https://github.com/vladikoff/chromeos-apk/blob/master/archon.md

Here is a tutorial how to run gReader

Note: You must have Chrome 37+ installed. (If it doesn't work try Chrome Canary).
OS X Note: Grab the latest version of Chrome Canary, Chrome won't work because 64 bit is required by the custom runtime.

  1. Download the runtime: https://bitbucket.org/vladikoff/archon/get/v1.1-x86_64.zip
  2. Unzip file and load the directory as an unpacked extension in Chrome
    Go to Extensions in Chrome and enable Developer mode. Press Load unpacked extensions... button and choose the unzipped directory
  3. Download gReader Chrome extension: https://github.com/noinnion/greader/tree/master/chrome
  4. Unzip file and load the directory as an unpacked extension in Chrome
  5. Click on Launch to load gReader.
