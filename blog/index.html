Customizable floating action button
29 Dec 2014

Since version 4.0.0 gReader introduced the new Material Design. One of its element is the floating action button (FAB) to mark all as read. To mark all articles as read you can swipe the button horizontal. Swiping vertical give you more additional options. With the update v4.0.2 we make it customisable. You can customise its position, style or opacity to your desire. All settings can be found in Settings->Reading Preferences->Mark All As Read.

Material Design in gReader

You might have heard about Material Design with the introduction of Lollipop. We take this occasion to redesign gReader (News+ coming soon) with the new Google design guideline. The new changes will be available in v4.0.0. https://play.google.com/store/apps/details?id=com.noinnion.android.greader.reader

Run gReader on ChromeOS or Chrome in Windows, MacOSX and Linux
20 Sep 2014

Google has officially made it possible to run Android apps on Chrome OS devices. Developer Vladikoff made ArChon, a customized version of the Android Runtime for Chrome, which loads up as a standard manual Chrome extension.

Feedly under DDoS attack
11 Jun 2014

Many of you have report us about synchronization issue with Feedly. The reason for the issue is that Feedly was attacked by criminals with a distributed denial of service attack (DDoS).
